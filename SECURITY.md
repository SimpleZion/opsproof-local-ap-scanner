# Security Policy

Do not post private finance, supplier, bank, tax, payroll, customer, or payment-card data in GitHub issues, pull requests, discussions, screenshots, or attachments.

For public issues, use one of these safer formats:

- Header-only CSV examples.
- Fake rows.
- Redacted rows.
- Synthetic supplier names.
- Screenshots with private values removed.

The scanner is designed to run locally in the browser and block network APIs after startup. If you find a path that sends user-loaded data over the network, report it as a security issue.
